VUES GESTION FINANCE
====================
Toutes les vues FXML du module Finance sont regroupées ici.

- FinanceView.fxml         → Admin : contrats, bulletins, primes, banques, génération PDF
- EmployeurFinanceView.fxml → Employé : Ma Paie + chatbot
- UserFinanceView.fxml     → Consultation utilisateur
- paiement.fxml            → Paiement avance projet (Stripe)

Chargement : MainView charge ces fichiers via /finance/views/NomFichier.fxml

Documentation complète (métiers avancés, API, emplacements) :
  → docs/FINANCE_ORGANISATION_ET_METIERS_AVANCES.md
