## 🚀 La Vision

**Skilora Tunisia** n'est pas seulement une plateforme ; c'est un pont vers l'avenir.

Dans un contexte où 41% de la jeunesse tunisienne fait face au chômage, Skilora émerge comme un écosystème vital conçu pour libérer le potentiel. Nous transformons le parcours de l'éducation à l'emploi en connectant les chercheurs d'emploi tunisiens qualifiés directement avec des opportunités de carrière mondiales.

En intégrant un matching intelligent des compétences, une vérification biométrique et une gestion de profil fluide, Skilora offre un marché de confiance et efficace pour que les candidats montrent leur vraie valeur et pour que les employeurs trouvent le candidat idéal—instantanément.

---

## 🌟 Pourquoi Skilora ?

*   **🌍 Portée Mondiale, Impact Local** : Ouvrir les portes aux marchés internationaux tout en cultivant les racines locales.
*   **🧠 Matching Intelligent** : Fini les candidatures sans but. Nos algorithmes assurent que les candidats voient les rôles pour lesquels ils sont vraiment qualifiés.
*   **🛡️ Confiance & Sécurité** : Avec la **Reconnaissance Biométrique Faciale**, nous garantissons que chaque profil est authentique, renforçant la confiance entre employeurs et candidats.
*   **💻 Écosystème Fluide** : Une architecture double-client comprenant une application Desktop JavaFX robuste et un portail Web Symfony (en développement).

## 📋 Contexte du Projet

*Développé avec passion par les étudiants d'**ESPRIT - Honoris United Universities**.*

Cette application constitue notre **Projet Intégré de Développement (PIDEV)**, démontrant une architecture moderne et évolutive qui résout des problèmes concrets.

**Module :** 3A - Web Java | **Durée :** Sprint 0-1 (Jan-Mar 2026) | **Équipe :** 6 Développeurs

### 🛠️ Modules Fonctionnels (Les 6 Gestions)

Le projet est divisé en 6 modules interconnectés, chacun géré par un membre de l'équipe :

1.  **Profils & Matching Intelligent** : IA de matching, extraction CV, gestion profils.
2.  **Formation & Certification** : Cours, quiz, certificats Blockchain.
3.  **Recrutement International** : Gestion offres, entretiens vidéo, fuseaux horaires.
4.  **Support & Assistant IA** : Chatbot intelligent, ticketing, base de connaissances.
5.  **Finances & Rémunération** : Paie multi-devises, calculs fiscaux (TN/UE).
6.  **Communauté & Networking** : Réseau social, mentorat, événements.

## 🔧 Installation & Configuration

### Prérequis

*   **Java JDK 17+**
*   **Maven 3.8+**
*   **MySQL Server**
*   **Python 3.x** (pour le module de reconnaissance faciale)

### Base de Données

Le projet utilise une base de données unifiée.
1.  Importez le fichier `skilora.sql` situé à la racine du projet dans votre serveur MySQL.
2.  Configurez vos identifiants dans `src/main/resources/database.properties` (si applicable) ou via l'interface de configuration au premier lancement.

### Lancement de l'Application

```bash
mvn clean javafx:run
```
