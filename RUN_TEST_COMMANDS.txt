===========================================
HOW TO RUN LOGIN TEST
===========================================

METHOD 1: Via IntelliJ IDEA Maven (Easiest)
---------------------------------------------
1. Open IntelliJ IDEA
2. Open the Maven tool window (View → Tool Windows → Maven)
3. Expand: skilora-tunisia → Plugins → exec
4. Double-click: exec:java
5. In the Run Configuration, set:
   - Main class: com.skilora.TestLogin
   - Or add to VM options: -Dexec.mainClass=com.skilora.TestLogin

OR use the Maven command directly in IntelliJ Terminal:
-------------------------------------------------------
Use IntelliJ's built-in terminal (Alt+F12) and run:

"C:\Program Files\JetBrains\IntelliJ IDEA 2025.2.2\plugins\maven\lib\maven3\bin\mvn.cmd" clean compile exec:java -Dexec.mainClass="com.skilora.TestLogin" -Dexec.classpathScope=compile

METHOD 2: Right-click in IntelliJ (Simplest)
---------------------------------------------
1. Navigate to: src/test/java/com/skilora/TestLogin.java
2. Right-click on the file
3. Select: "Run 'TestLogin.main()'"

This will run it directly with all dependencies loaded.

METHOD 3: Create Run Configuration
---------------------------------------------
1. Run → Edit Configurations
2. Click "+" → Application
3. Name: "Test Login"
4. Main class: com.skilora.TestLogin
5. Use classpath of module: skilora-tunisia
6. Click OK and Run

===========================================
WHAT THE TEST WILL DO
===========================================

The TestLogin script will:
1. Test login for: admin/admin123, user/user123, employer/emp123
2. Check if accounts are locked and auto-reset them
3. Verify users exist in database
4. Check password hash types (BCrypt vs Plaintext)
5. Test password verification
6. Test actual login via AuthService
7. Show detailed results for each account
